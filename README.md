# flows

Prefect Orion examples 

## Getting started

Prerequisites:

- make
- node (required for pyright)
- python >= 3.7
- docker compose
- k3d to create a k3s kubes cluster
- sqlite 3

To start:

- Install the [development environment](CONTRIBUTING.md#Development-environment): `make install`
